CountryFinder
Met deze applicatie kan je de informaties van een land vinden.

FindByPrefix: via deze pagina kan je een land vinden via de Prefix(bv: Fr) of de naam van een land(bv:Frankrijk). Als je op de naam of de wikiDataID klikt, dan krijg je meer details van het land. Op deze pagina worden verschillende functionaliteiten van REST API toegepast, zoals Hypermedia links, paging, search en opvragen van een resource via de ID...

CountryDetail: op deze pagina kan je de details vinden van een land via ID(bv:Q142 van frankrijk).
